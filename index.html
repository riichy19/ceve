Sobre mí

Soy un desarrollador tranquilo, con pensamiento sistemico y compromiso por entregar soluciones elegantes que resuelven problemas reales. Disfruto diseñar arquitecturas modulares, documentar buenas practicas y acompaniar la adopcion tecnologica con equipos multidisciplinares.

  • Full Stack: Node.js, Angular, React, PHP, Java (patrones MVC y REST)
  • Datos y automatización: pipelines con Node, SFTP y Amazon S3; CSV y conversión de audio
  • Seguridad: autenticación con 2FA (Angular + Node), validación de RFC, correo y teléfono
  • Reportería: SQL Server para reportes y exportación a CSV; limpieza y clasificación por año/volumen
  • Operaciones: despliegues internos, tareas programadas, bitácoras y alertas SMS; monitoreo y estabilidad
  • Documentos en lote: Angular + Node con LibreOffice/Easy_Template; generación masiva de PDF con validaciones

En resumen, desarrollo software web full stack con Node.js, Angular, PHP y Java; conecto con bases de datos SQL y NoSQL; construyo módulos de autenticación con 2FA; genero reportes y automatizo procesos sobre SQL Server; realizo despliegues internos con monitoreo y alertas; manejo infraestructura básica (SFTP, S3, servidores locales) y también doy soporte técnico a usuarios y equipos.
